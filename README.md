# 求人票抽出デモ（実務向けプロトタイプ）

ハローワーク系の求人票を想定し、**テキスト / PDF / URL** の3つの入力方法から、指定項目を見出し付きで1行ずつ整理して表示するフロントエンドデモです。

## 対応している入力方法

1. 求人票テキストを貼り付け
2. PDFファイルをアップロード（ブラウザでテキスト抽出できるPDFのみ）
3. 求人URLを入力（取得できない場合の案内付き）

## 使い方

1. `index.html` をブラウザで開く
2. 入力方法タブ（テキスト/PDF/URL）を選ぶ
3. データを入力して「抽出する」を押す
4. 抽出結果を確認し、「結果をコピーする」でそのままコピー
5. 必要なら「sample.txt を読み込む」でサンプルを入力欄へ読み込み

## 主な仕様

- 見出し語ベースの簡易抽出（厳密な自然言語解析は未実装）
- 未抽出項目は `詳しくは面談時にお問合せください。` を表示
- キャッチコピーは20〜120文字で仮要約
- 特長①〜③は20文字以内で仮生成
- 本文は150〜300文字で仮要約
- 給与は固定フォーマット（【月給】、【内訳】、昇給、賞与）で整形

## URL取得の制約

- 求人サイト側のCORS、bot対策、認証、robots制限などにより本文取得できない場合があります。
- 取得失敗時は画面に「本文を貼り付けてください」と案内を表示します。

## PDF対応の制約

- テキスト情報を持つPDFのみ抽出可能です。
- 画像PDF、スキャンPDF、保護PDFは抽出失敗することがあります。
- ブラウザ環境やネットワーク状況によって PDF ライブラリ読み込みに失敗する場合があります。

## sample.txt

- `sample.txt` に、指定項目を含むサンプル求人票を同梱しています。

## 今後のLLM連携案

1. 見出し抽出（ルールベース）→欠損項目のみLLM補完
2. 本文・特長・キャッチコピーをLLMで品質改善
3. 給与整形の妥当性チェック（単位・範囲・矛盾）をLLM + ルールで二重検証
4. URL先HTMLの重要箇所抽出をLLMで支援し、サイトごとの差異に追随


## 前処理（精度改善）

- PDF/URL入力では抽出前に `preprocessText(text, sourceType)` を実行しています。
- 主な処理:
  - `<br>` / `<br />` を改行へ変換
  - `Title:` / `ID:` / 投稿日 / 更新日 などのノイズ行を除去
  - URL由来のHTMLタグ断片（`<...>`）の除去
  - 連続空白・過剰改行の整理
  - 見出し候補（事業所名、職種、雇用形態、仕事内容、給与、勤務時間、休日、待遇、試用期間など）の前に改行挿入
- 画面のデバッグ表示で「生テキスト」と「前処理後テキスト」を比較確認できます。


## 媒体別抽出（現状対応）

- 対応媒体は以下の2つのみです。
  1. ハローワークPDF
  2. ジョブメドレーURL（`job-medley.com`）
- その他媒体は「未対応」と表示し、抽出は行いません。
- 媒体ごとに `preprocess` と `extract` を分離しています。
  - ハローワークPDF: 帳票見出し（求人番号/職種/雇用形態 等）に寄せた前処理
  - ジョブメドレーURL: HTML断片除去とサイト特有見出し（法人・施設名/募集職種 等）に寄せた前処理
- デバッグ表示で以下を確認できます。
  - 媒体判定結果
  - 生テキスト
  - 前処理後テキスト


## ハローワークPDF（パートタイム）専用ルール

- `preprocessHelloWorkPdf(text)` で縦書き崩れを補正し、主要ラベル（事業所名/仕事内容/雇用形態/試用期間/時間外労働/休日等/必要な経験・知識・技能等/必要な免許・資格）を改行正規化します。
- セクション見出し（1〜7）で本文を分割し、セクション単位で抽出します。
  - 1 求人事業所
  - 2 仕事内容
  - 3 賃金・手当
  - 4 労働時間
  - 5 その他の労働条件等
  - 6 会社の情報
  - 7 選考等
- 時給求人では「時間額（a+b）」を `【時給】` として優先採用し、`基本給（時間換算額）` と `その他手当付記事項` を内訳に反映します。
- 本文 / キャッチコピー / 特長①〜③は、仕事内容・会社の特長・求人に関する特記事項を優先して生成します。
- デバッグ表示では「生テキスト」「前処理後テキスト」「セクション分割結果」を確認できます。


## ハローワークPDFで安定しやすい項目 / 不安定な項目

### 安定しやすい項目
- 事業所名
- 職種
- 雇用形態
- 就業時間（勤務時間）
- 時間外労働
- 受動喫煙対策

### 不安定な項目
- 就業場所の変更範囲（記載が省略されることが多い）
- 試用期間の待遇・その他（表現揺れが大きい）
- 休日・休暇（週休二日制/シフト/労働日数の分散記載）
- 待遇（通勤手当/駐車場/保険/託児施設/正社員登用が散在）
- 応募資格（学歴・経験・資格が別行で崩れる場合）

※ 不安定項目はデバッグ表示の「セクション分割結果」「未抽出項目の探索ログ」で確認し、ルール調整してください。
