# 求人票抽出デモ

求人票テキストを貼り付けると、見出しごとに情報を整理して表示するシンプルなフロントエンドデモです。  
サーバー不要で、`index.html` をブラウザで開くだけで動作します。

## ファイル構成

- `index.html` : 画面レイアウト
- `style.css` : スタイル
- `script.js` : 抽出ロジック（JavaScriptのみ）
- `sample.txt` : サンプル求人票
- `stylesheet.css` : 既存ファイル（今回のデモでは未使用）

## 使い方

1. `index.html` をブラウザで開く
2. 必要に応じて `sample.txt` を開き、内容をテキストエリアに貼り付ける
3. 「抽出する」ボタンを押す
4. 抽出結果エリアに各項目が表示される
5. 「結果をコピーする」ボタンで、抽出結果をテキスト形式でコピーできる

## 抽出対象項目

- キャッチコピー
- この求人の特長はココ①
- この求人の特長はココ②
- この求人の特長はココ③
- 本文
- 職種
- 雇用形態
- 仕事内容
- 応募資格
- 勤務地
- アクセス
- 勤務時間
- 休日
- 給与
- 待遇・福利厚生
- 試用期間
- 受動喫煙対策

## 現時点の実装方針（簡易版）

- 厳密な自然言語解析は行わず、見出し語（例: `仕事内容`, `給与` など）に反応して値を取得
- 見出し行に値が無い場合、次行以降を次の見出しまで取得
- 見つからない項目は `詳しくは面談時にお問合せください。` を表示
- キャッチコピー、特長①〜③、本文は入力全体や一部行から簡易要約して仮生成

## 現時点の限界

- 見出し表記ゆれ（例: 独自名称、記号付き見出し）に弱い
- 求人票のレイアウト次第で、本文や特長の要約品質が不安定
- 箇条書き・表形式・複数求人混在テキストの精度保証は未対応
- ブラウザの設定によってはクリップボード書き込みが制限される場合がある
